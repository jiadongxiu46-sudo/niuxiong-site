Anthropic 5/28-5/29
Claude Opus 4.8发布+完成650亿Series H——Anthropic进入全速扩张期
Anthropic 5/28发布Opus 4.8：常规API价格不变（输入$5/输出$25），新增effort控制、Claude Code dynamic workflows和更便宜的fast mode。同期完成650亿美元Series H融资，投后估值9650亿美元，成为AI领域史上最大单轮融资之一。
这是典型的双面信号：Opus 4.8价格不变说明Anthropic在打规模战，不靠单价赚钱，靠跑量占领工作流；650亿融资说明资本市场愿意为这场持久战买单。Claude Code正在变成开发者工作流的默认选项，这是比模型参数更重要的护城河。

这周最重要的AI信号不是谁发了新模型，而是三家头部厂商同时押注同一个方向：编码代理。这不是巧合，是集体判断。谁先把代理做成可计费、可治理、可接进企业工作流的产品，谁就拿到下一个周期的入口。

地缘冲突
霍尔木兹 全周持续
美伊停火谈判悬而未决，美军继续拦截商船——最重要的跨资产变量
本周中东最重要主线：美国对霍尔木兹相关伊朗机构连续制裁（5/27、5/28），布伦特油价从104美元一度跌至91美元，但停火协议至今未落地。5/30，美军在阿曼湾导弹拦截一艘试图突破封锁的商船，外交未拍板、军事不停手的格局贯穿全周。
停火预期是这周美股创新高的最大外部支撑，但这是一个可以瞬间逆转的变量。白宫若否掉协议，油价和美债会在同一天重定价。这是目前所有风险资产最需要盯住的单一地缘节点。
以黎/俄乌 全周
以军推近贝鲁特+俄无人机击中罗马尼亚——两条线同步升温
以色列把打击从南黎推进至贝鲁特南郊（5月6日停火后首次），以黎军方5/29在五角大楼举行首次直接会谈，但地面推进未停。俄无人机碎片5/29落入罗马尼亚（北约领土）住宅楼，俄方拒绝美方对基辅的警告。泽连斯基警告俄罗斯正准备新一轮大规模攻势。
俄乌碎片落在北约领土是本周升级风险中被低估的一条——不是直接攻击，但每一次这样的事件都在测试北约集体防御的底线。这不是会让市场立刻反应的新闻，却是会让地缘风险溢价长期抬高的积累。

宏观 5/28数据
PCE未超预期+停火延长压低油价，给科技估值撑起两周缓冲窗口
4月核心PCE环比0.2%低于预期0.3%，Q1 GDP二读下修至1.6%；市场解读为降息友好，但美联储仍处于不鸽不鹰的政策真空期。本周最重要的跨资产信号不是通胀数据，而是霍尔木兹停火预期把布伦特油价从104美元压回91美元——这一个动作同时压住了能源通胀和长端收益率，撑起了全周科技股的估值空间。
油价回落是本周美股创新高的真正外部条件，不是AI。这也意味着这套逻辑的脆弱性很清晰：霍尔木兹一旦出事，油价涨，长端利率跟，科技估值第一个受压。

宽度 全周观察
指数连创新高，但涨跌比告警——龙头驱动而非全面普涨
本周涨跌比持续偏窄：Nasdaq成分股涨跌比5/29仅1.17:1，Alphabet单日-2.5%，能源板块全周-0.89%，小盘Russell 2000明显跑输大盘。简单说：指数在创新高，但创新高的只是前10只股票。
宽度不够是这轮行情最需要监控的结构性弱点。当少数龙头独撑指数，容错率就会低于指数表面强度——任何一只龙头的意外（财报miss/监管/地缘），冲击会被杠杆放大。

BTC从73k回到108k，但ETF全周都在流出——这是本周加密市场最需要解释的悖论。价格涨靠的是存量筹码坚守，不是新钱进场。机构资金用ETF投票：他们还没回来。
监管制度 本周
CFTC批准Kalshi永续+Kraken推出受监管永续，制度建设一周两步
CFTC本周批准Kalshi上线BTC永续期货，并给出Coinbase/Deribit路径的no-action口径（5/29）；Kraken宣布推出首个CFTC监管下的加密永续期货（5/30）；OKX将于6/6修订资金费率公式，Bybit将于6/11修订OI统计口径（展示值将约降50%，注意口径变化勿误读为去杠杆）。
监管制度这周走了两步，方向是好的——合规衍生品入场通道在打开。但Kraken同期下架了21个代币，含多个AI相关资产，说明合规化是双向的：入口开了，不合格的先出去。监管周期里，有标准的资产和没标准的资产，最终会出现巨大的流动性分化。

本期结论
本周市场用行动回答了一个问题：AI主线能不能从英伟达扩散出去。答案是可以——微软的平台层、戴尔的服务器层、Salesforce的应用层，本周同时兑现。

但支撑这一切的外部条件是脆弱的：霍尔木兹停火预期把油价从104压到91，这一个动作撑住了PCE、撑住了长端利率、撑住了科技估值。白宫一旦否掉协议，这条链条会反向传导。

加密这周最割裂：BTC从73k回到108k，但ETF全周净流出5.7亿。价格靠存量筹码在撑，机构还没回来。下周真正重要的变量只有三个：霍尔木兹白宫能否拍板，CrowdStrike/HPE/Broadcom财报能否验证AI继续扩散，BTC ETF能否从净流出转正。
